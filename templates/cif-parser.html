CIF Parser

A CIF Parser is a class that converts information from the Client into data in our database

The CIF Parser will be called during a batch job it is not part of the web flows.

A client uses this batch job to update our database with liabilities. Basically who owes what by what date.

CIF Parsers input files (CIFS) are sent to us in a Width Based Format or a Comma Seperated Format

CIF Parser - Testing

To Test a CIF Parser we user the Java Class:

- TestLiabilityFileLoad

You will need a sample file from the biller to test, place the file at the following location:

- /paybox/test/com/paymentus/test/paybox/batch/liability/abc/liability-sample.csv

replace the abc in this example to the TLA being worked on e.g. abc, btxu, bwwp, etc.

In this class you should set the client variable to the TLA of the client

Run the test class and your output should be in paybox's output folder
